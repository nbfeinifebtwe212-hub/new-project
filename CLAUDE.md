# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## プロジェクト概要

チャットボット付きホームページの構築プロジェクト。モダンで洗練されたデザインと、スマートフォンを含む全デバイスへのレスポンシブ対応が必須要件。

## 基本ルール

- **返答は日本語で行う**
- **コードのコメントは日本語で書く**
- **デザインはモダンで洗練されたものにする**
- **レスポンシブ対応（スマホでも見やすい）を必ず行う**

## デザイン方針

- モバイルファーストアプローチで実装する
- ブレークポイントは `sm: 640px` / `md: 768px` / `lg: 1024px` を基準とする
- アニメーションや遷移はスムーズかつ控えめに（UX を損なわない程度）
- チャットボットUIはページの邪魔にならないよう、フローティング形式を優先する

## チャットボット実装方針

- チャット UI はページ右下にフローティングボタンで配置する
- メッセージの送受信はストリーミング表示を推奨する
- Claude API（`claude-sonnet-4-6` モデル）を使用する
- API キーは環境変数（`.env`）で管理し、コードに直接書かない
